CONNECT FOUR GAME 

A classic Connect Four game built using pure HTML, CSS, and JavaScript.
This project focuses on implementing core game logic, DOM manipulation,
and a clean user interface without using any external libraries or frameworks.

This project was developed as part of an InternPe Web Development Internship task.

--------------------------------------------------

FEATURES

- Two-player gameplay
- Turn-based disc dropping
- Automatic win detection (horizontal, vertical, diagonal)
- Game reset functionality
- Simple and user-friendly interface
- Built using only vanilla JavaScript

--------------------------------------------------

TECH STACK

HTML – Structure of the game board
CSS – Styling and layout
JavaScript – Game logic and interactivity

No frameworks. No libraries. Only core web fundamentals.
